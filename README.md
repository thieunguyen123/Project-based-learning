## Usage:-

- Clone my repository.
- Open CMD in working directory.
- Run `Food_app.py`.
- Go to the `http://127.0.0.1:5000` and test it. It is a local Flask App.
- Heroku App is available here:- [Food Prediction App](https://food-ml.herokuapp.com/)
